PDF to PNG Converter
===================

Bu program, PDF dosyalarınızı PNG formatına dönüştürmenizi sağlar.

Kurulum:
--------
1. ZIP dosyasını bilgisayarınıza indirin
2. ZIP dosyasını açın
3. PDF_to_PNG_Converter.exe dosyasını çalıştırın

Kullanım:
--------
1. Program açıldığında, DPI değerini seçin (varsayılan: 100)
2. "Klasör Seç" veya "ZIP Dosyası Seç" butonlarından birini kullanarak:
   - PDF dosyalarının bulunduğu klasörü seçin
   - VEYA
   - PDF dosyalarını içeren bir ZIP dosyası seçin
3. "Dönüştür" butonuna tıklayın
4. İşlem tamamlandığında, PNG dosyaları otomatik olarak kaydedilecektir

Özellikler:
----------
- PDF dosyalarını PNG formatına dönüştürme
- Klasör veya ZIP dosyası desteği
- Özelleştirilebilir DPI ayarları (72-600 DPI)
- İlerleme takibi ve hata raporlama
- Orijinal dosya yapısını koruma
- Birden fazla PDF dosyasını işleme

Gereksinimler:
-------------
- Windows 7 veya daha yeni bir işletim sistemi
- Ek yazılım gerektirmez

Not: Program ilk açılışta Windows Defender veya antivirüs programınız tarafından engellenebilir. 
Bu durumda "Daha fazla bilgi" seçeneğine tıklayıp "Yine de çalıştır" butonunu kullanabilirsiniz.

Sorun yaşarsanız veya yardıma ihtiyacınız olursa, lütfen iletişime geçin.
